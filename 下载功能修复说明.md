# 下载功能修复说明

## 问题描述
点击下载按钮后，显示 JSON 错误页面而不是下载文件：
```json
{
  "message": "File not found",
  "success": false
}
```

## 问题原因
1. **文件路径存储问题**: 上传报表时，`file_path` 字段保存的是相对路径
2. **路径转换问题**: 在检查文件是否存在时，没有将相对路径转换为绝对路径
3. **文件保存问题**: `file.save()` 需要绝对路径才能正确保存文件

## 修复内容

### 1. 修复上传逻辑 (`app/services/report_service.py`)
```python
# 使用绝对路径保存文件
upload_folder = current_app.config.get('UPLOAD_FOLDER', 'uploads')
base_upload_dir = os.path.join(current_app.root_path, '..', upload_folder)
base_upload_dir = os.path.abspath(base_upload_dir)

customer_folder = os.path.join(base_upload_dir, f'customer_{customer_id}')
os.makedirs(customer_folder, exist_ok=True)

# 保存文件 - 使用绝对路径
abs_file_path = os.path.join(year_month_folder, unique_filename)
file.save(abs_file_path)

# 保存相对路径到数据库（便于迁移）
relative_path = os.path.join(f'customer_{customer_id}', f'{year}', f'{month:02d}' if month else '00', unique_filename)

# 创建报表记录时保存相对路径
report = Report(
    ...
    file_path=relative_path,
    ...
)
```

### 2. 修复下载逻辑 (`app/services/report_service.py`)
```python
@staticmethod
def get_report_file_path(report_id: int) -> Tuple[Optional[str], Optional[str]]:
    """获取报表文件路径"""
    from flask import current_app
    
    report = Report.query.get(report_id)
    if not report:
        return None, _('Report not found')
    
    if not report.file_path:
        return None, _('File path not recorded')
    
    # 如果是相对路径，转换为绝对路径
    file_path = report.file_path
    if not os.path.isabs(file_path):
        upload_folder = current_app.config.get('UPLOAD_FOLDER', 'uploads')
        file_path = os.path.join(current_app.root_path, '..', upload_folder, report.file_path)
        file_path = os.path.abspath(file_path)
    
    # 检查文件是否存在
    if not os.path.exists(file_path):
        current_app.logger.error(f'File not found: {file_path}')
        return None, _('File not found')
    
    return file_path, None
```

## 已上传文件的处理

对于已经上传的报表文件，如果 `file_path` 字段存储的是绝对路径，下载功能应该仍然有效。

如果存储的是不正确的路径，需要：
1. 手动更新数据库中的 `file_path` 字段
2. 或者重新上传这些文件

## SQL 更新示例（如果需要修复旧数据）
```sql
-- 查看当前的文件路径
SELECT id, report_name, file_path FROM reports;

-- 如果需要更新（示例）
UPDATE reports 
SET file_path = REPLACE(file_path, '旧路径', '新路径')
WHERE file_path LIKE '旧路径%';
```
